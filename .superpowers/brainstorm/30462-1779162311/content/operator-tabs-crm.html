Design Notes

  • Score badge colors: Green (70-100), Amber (40-69), Red (0-39) — semantic meaning at a glance
  • Status dot: Green = active, Gray = inactive, Yellow = pending — consistent with existing app
  • Avatar initials: Same pattern as existing ProfileView — gradient background, white text
  • Filter chips: Match existing MessagesListView filter pattern (horizontal scroll, Theme.primaryGradient selected)
  • Cards: 14pt corner radius (Theme.cornerMD), Theme.cardShadow(), white background
  • Tab bar icons: SF Symbols (person.2.fill, megaphone.fill, message.fill, person.fill) — NOT emojis (shown here for HTML only)
