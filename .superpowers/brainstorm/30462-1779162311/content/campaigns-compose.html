Design Notes

  • Campaign status badges: ENVIADA = green, RASCUNHO = amber with left border accent
  • Draft cards: Show "Enviar Agora" + "Editar" action buttons inline
  • Sent cards: Show message count + date + sender info in footer
  • Compose Message sheet: Segmented control toggles between "Cliente" (1:1) and "Segmento" (broadcast). Fields: recipient picker, title, body, optional image URL + action buttons
  • Create Campaign sheet: Name, segment picker, message content (title, body), optional deeplink. Two actions: "Salvar Rascunho" (creates DRAFT) or "Criar e Enviar" (creates + immediately sends via POST /campaigns/{id}/send)
  • "Criar e Enviar" uses campaignGradient (orange→red) to visually distinguish from normal primary actions — signals "this will send immediately"
  • Optional fields: Image URL and action buttons collapsed behind chips to reduce form noise (progressive disclosure)
  • All sheets: .presentationDetents([.large]), drag handle, swipe to dismiss
